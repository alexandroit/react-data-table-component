# React Data Table Component

[![GitHub release](https://img.shields.io/github/v/release/alexandroit/react-data-table-component?display_name=tag&style=flat-square)](https://github.com/alexandroit/react-data-table-component/releases/) [![GitHub stars](https://img.shields.io/github/stars/alexandroit/react-data-table-component.svg?style=social&label=Star&maxAge=2592000)](https://github.com/alexandroit/react-data-table-component/stargazers/)

This maintained `@revivejs` release keeps the original declarative table API while updating the package metadata, documentation, and publishing workflow for the current repository owner.

If you want a flexible data table without adopting an entire spreadsheet-style grid framework, this package gives you sorting, selection, expandable rows, pagination, theming, and custom cell rendering with a lightweight setup.

# Key Features

- Declarative configuration
- Built-in and configurable:
  - Sorting
  - Selectable Rows
  - Expandable Rows
  - Pagination
- Themeable/Customizable
- Accessibility
- Responsive (via horizontal scroll and flex layout)
